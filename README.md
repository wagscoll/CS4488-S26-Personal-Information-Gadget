# CS4488-S26-Personal-Information-Gadget


After pulling the repository, run the CLI demo by inputting:
```
cd "src/Demo"
dotnet build
dotnet run
```

Run the Windows Form Budget and Shopping List App demo by inputting:
```
cd "src/FormApp/BudgetFormApp"
dotnet build
dotnet run
```

Run the Windows Form Project and Task Manager App demo by inputting:
```
cd "src/FormApp/ProjectAndTaskManagerFormApp"
dotnet add package DocumentFormat.OpenXml
dotnet build
dotnet run
```
